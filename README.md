## 🛠️ Cài Đặt

### Cách 1: Cài đặt từ File JSON (Đề xuất)

1. **Tải file Registry** [`vbook plugin.json`](https://raw.githubusercontent.com/Laodai123/vbook-plugins-collection/main/vbook%20plugin.json) về máy
2. Mở ứng dụng **vBook** trên điện thoại
3. Vào **Cài đặt → Quản lý Plugin → Nhập từ File**
4. Chọn file `vbook plugin.json` vừa tải
5. Chọn plugin bạn muốn cài đặt từ danh sách
6. ✅ Hoàn tất! Bắt đầu đọc truyện ngay

### Cách 2: Cài đặt từng Plugin riêng lẻ

1. Mở thư mục plugin bạn muốn trong repository
2. Copy **đường dẫn raw** của file `plugin.json` trong thư mục đó
   ```
   https://raw.githubusercontent.com/Laodai123/vbook-plugins-collection/main/[Author] - PluginName/plugin.json
   ```
3. Trong vBook, chọn **Thêm Plugin → Nhập URL**
4. Paste đường dẫn trên vào
5. ✅ Plugin sẽ được cài đặt tự động

### Cách 3: Tải file .zip

1. Mỗi plugin đều có sẵn file `.zip` trong thư mục
2. Tải file `.zip` về máy
3. Trong vBook, chọn **Nhập từ File → Chọn file .zip**

### ⚙️ Cấu trúc Plugin

Mỗi plugin trong collection có cấu trúc chuẩn:

```
[Author] - PluginName/
├── plugin.json      ← File cấu hình plugin (bắt buộc)
├── icon.png         ← Biểu tượng plugin
├── src/
│   ├── config.js    ← Cấu hình nguồn
│   ├── home.js      ← Trang chủ
│   ├── search.js    ← Tìm kiếm
│   ├── detail.js    ← Chi tiết truyện
│   ├── toc.js       ← Mục lục
│   ├── chap.js      ← Nội dung chương
│   ├── gen.js       ← Phân loại
│   └── genre.js     ← Thể loại
└── Author - PluginName.zip  ← File nén sẵn sàng cài
```

## 📜 File Registry (vbook plugin.json)

File [`vbook plugin.json`](https://raw.githubusercontent.com/Laodai123/vbook-plugins-collection/main/vbook%20plugin.json) là file registry chính thức chứa thông tin của tất cả **398 plugin**.

**Cấu trúc mỗi item:**

```json
{
  "name": "Tên Plugin",
  "author": "Tên tác giả",
  "path": "URL raw plugin.json",
  "version": 50,
  "source": "https://nguon-truyen.com",
  "icon": "URL icon.png",
  "description": "Mô tả plugin",
  "type": "novel | chinese_novel",
  "locale": "vi_VN | zh_CN | en_US"
}
```

## 🤝 Đóng Góp

Chúng tôi hoan nghênh mọi đóng góp từ cộng đồng!

### 📋 Cách Đóng Góp

1. **Fork** repository này
2. Tạo thư mục mới theo format: `[TênBạn] - TênPlugin/`
3. Tạo file `plugin.json` với cấu trúc đúng
4. Thêm source code vào thư mục `src/`
5. Thêm file `icon.png` (tùy chọn)
6. Tạo file nén `.zip`
7. Cập nhật file `vbook plugin.json` ở thư mục gốc
8. Tạo **Pull Request**

### ✅ Yêu Cầu Khi Đóng Góp

| Yêu cầu | Chi tiết |
|---------|---------|
| `plugin.json` | Phải đúng format JSON, không có lỗi syntax |
| `src/` | Chứa các file JS theo chuẩn VBook Plugin API |
| Tên thư mục | Format: `[TênTácGiả] - TênPlugin` |
| PR | Mô tả rõ plugin hỗ trợ nguồn nào |
| Test | Kiểm tra plugin hoạt động trước khi submit |

### 🧪 CI/CD

Repo sử dụng **GitHub Actions** để tự động kiểm tra:

- ✅ Kiểm tra syntax JSON của tất cả `plugin.json`
- ✅ Thống kê số lượng plugin
- ✅ Chạy tự động khi có Push hoặc Pull Request
