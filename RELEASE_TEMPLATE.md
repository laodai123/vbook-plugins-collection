## 📥 Download Instructions

### For Users

1. **Update via VBook App**
   - Open VBook application
   - Go to Settings → Plugins
   - Tap "Update All" or update individual plugins

2. **Manual Download**
   - Download the latest `plugin.json` from the [Releases](https://github.com/Laodai123/vbook-plugins-collection/releases) page
   - Place it in your VBook plugins directory

### For Developers

```bash
# Clone the repository
git clone https://github.com/Laodai123/vbook-plugins-collection.git

# Update to latest version
```
